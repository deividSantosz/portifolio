SOBRE SOBRE SOBRE SOBRE SOBRE SOBRE

Como você ja viu, eu sou o Deivid, tenho 24 anos, sou estudante de Sistemas de Informação desde 2021 na Universidade Estadual do Sudoeste da Bahia (UESB). Desde que comecei a cursar me dedico diariamente ao aprimoramento de minhas habilidades em desenvolvimento WEB, com foco no back-end e na criação de aplicações Android. Apaixonado por tecnologia, busco aprender continuamente, explorando ferramentas como Java, Spring Boot e Kotlin, para projetar soluções eficientes e inovadoras. Meu objetivo é transformar ideias em aplicações práticas, contribuindo para projetos que impactem positivamente o mundo ao nosso redor.

Conheça as tecnologias das quais tenho dominio!

As tecnologias listadas abaixo são tecnologias das quais tenho dominio e uma experiência maior em desenvolvimento.

Linguagem Java

Amplo conhecimento em desenvolvimento utilizando Java no back-end e no Android, incluindo conceitos avançados de POO, estrutura de dados, frameworks como Spring Boot, design Android com XML

Framework Spring

Amplo conhecimento em desenvolvimento utilizando o ecossistema Spring, incluindo Spring Boot para construção de APIs RESTful, Spring Security para autenticação e autorização, Spring Data para integração com bancos de dados.

Linguagem Kotlin

Experiência no desenvolvimento com Kotlin, aproveitando suas características modernas como sintaxe concisa, null safety e interoperabilidade com Java. Minha experiência com Kotlin foi apenas no Android.

Banco de dados PostgreSQL

Experiência com PostgreSQL, um banco de dados relacional de código aberto robusto e altamente escalável. Tenho familiaridade com sua utilização em ambientes de produção.

HTML 5

Experiência sólida em HTML, a base para a construção de páginas web. Familiaridade com tags modernas para uma melhor organização do conteúdo e proporcionar a melhor experiência para o usuario.

CSS 3

Experiência prática com CSS, utilizando suas propriedades para criar layouts responsivos e visualmente atrativos. Familiaridade com as melhores práticas, como Flexbox e Grid, para construção de layouts modernos e dinâmicos.

Git

Experiência com Git para controle de versão e colaboração em equipe. Utilizo Git para gerenciar repositórios de código, realizar commits claros e organizar fluxos de trabalho com branches, como Git Flow.

Projetos

Café na porta


Café na Porta é um aplicativo de delivery de café, permitindo que os usuários naveguem por uma variedade de opções de café, filtrem por preço e data, adicionem favoritos e mantenham um perfil personalizado. Este projeto foi desenvolvido com o objetivo de fornecer uma experiência de usuário agradável e intuitiva.

Back-end Login Page

O projeto consiste em construir um back-end para uma pagina de login e registro, utilizando das tecnologias de autenticação com JWT Token e Spring Security.

Clone Spotify

Essa aplicação foi construída durante a imersão Front End da Alura que aconteceu de forma online entre os dias 22 a 26 de janeiro de 2024. A aplicação consiste em fazer um clone do spotify e utilizar uma "fake api" para simular uma busca de artistas.

Web Services

A API consiste em aplicar os conceitos da arquitetura REST em um modelo de dominio com usuario, produtos, item, categorias, item do pedido e pagamento.
